[charter-flights-react](https://williamburgeson.github.io/charter-flights-react/) (repo at [https://github.com/williamBurgeson/charter-flights-react](https://github.com/williamBurgeson/charter-flights-react))

This vite template based react app is designed to model a modern react app of moderate complexity to show upcoming charter flights as well as the ability to select airports all over the world using a leaflet driven map.

This app is an ongoing project; when complete the intention is to allow the user to create and fully search all flights in the system. The conceptual idea is to evoke the functionality of reserving seats on a flight which we are all familiar with, but without the functional complexity of the extra logical layer of airports -> flights -> seats. Imagine yourself wealthy enough to be able to reserve the entire plane, rather than having to select seats on existing flights! Sadly this will remain a fantasy for the user to indulge their own imagination in, while allowing the functionality to pretend to do so...

More information in the readme in the repo
